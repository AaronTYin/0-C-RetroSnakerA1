# O-C-RetroSnakerA1
TanChiSheA1 with C\C++ and few WindowsAPI

>这算是做的第一个比较成型的小游戏了，基本功能实现的差不多，缺陷就是没法自己持续移动。

>此版本不再更新，之后将用其他语言重做。

>详见：[O_C++命令行贪吃蛇](https://mengze.top/O_CPP命令行贪吃蛇/)
